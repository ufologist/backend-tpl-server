后端模版页面服务器, 使用的后端模版引擎是: [Velocity](http://velocity.apache.org/), 适合前后端一起来协作编写页面

## 使用说明

* 安装 [Node.js](http://nodejs.org/) `6.x`
* 将 [fe](https://github.com/ufologist/backend-tpl-server/tree/master/fe) 目录中的文件复制到你的 Java Web 项目根目录的 fe 目录中
